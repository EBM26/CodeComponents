Array Filter Method

Title: Array.filter

Short Summary: method that creates a new array with all elements that pass the test implemented by the provided function (You are creating a new array based on the standards you want, filter is the "bouncer"). DOES NOT MUTATE ARRAY ITSELF.

Parameters/Arguments: 3 arguments: takes in the value(mandatory) of the item, the index(optional) of the item, and the array(optional) itself.

Examples

// return all the even numbers

const numbers = [1, 2, 3, 4, 5, 6]; const even = numbers.filter(isEven); function isEven(val) { return val % 2 === 0; }
