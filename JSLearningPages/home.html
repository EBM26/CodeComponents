Search

  1. Array.filter: array method that creates a new array with all elements that pass the test implemented by the provided function (You are creating a new array based on the standards you want, filter is the "bouncer"). DOES NOT MUTATE ARRAY ITSELF.
  2. Array.find: array method that finds the value of the first element in an array which matches the condition placed in the callback argument. Returns undefined if no match is found. DOES NOT MUTATE ARRAY ITSELF
  3. Array.findIndex: array method that finds the index (position) of the first element in an array which matches the condition placed in the callback argument. Returns -1 if no match is found. DOES NOT MUTATE ARRAY ITSELF
  4. Array.forEach: array method that executes a provided function once for each array element. DOES NOT MUTATE ARRAY ITSELF
  5. Array.map: array method that creates a NEW ARRAY populated with the results of calling a provided function on every element in the calling array. DOES NOT MUTATE ARRAY ITSELF
  6. Array.reduce: array method that executes a reducer function for each value of an array. It returns a single value which is the function's accumulated result. DOES NOT MUTATE ORIGINAL ARRAY
  7. Array.slice: array method that returns selected elements in an array, as a new array. Selects from a given start, up to a given end. DOES NOT MUTATE ORIGINAL ARRAY
  8. Array.sort: array method that sorts the elements of an array, overwrites the original array, sorts the elements as strings in alphabetical (converts everything to Strings) and ascending order (UTF-16 code units order). MUTATES ORIGINAL ARRAY
  9. Array.splice: array method that adds and/or removes array elements. MUTATES ORIGINAL ARRAY (which is one of the things that makes it different from slice)
