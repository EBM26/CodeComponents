Fundamentals of React.js Notes

React.js Basics

  • React.js is a library (as opposed to a framework) that allows you to create dynamic components that can interact with each other and with the user in your application/website.
  • First the React app and all other dependencies wanted are installed into the program using the terminal.
  • Then you import React and all the dependencies neede for that page.
  • Then you create a React component using the es6 class extender method that lets you extend the object and all it's properties
