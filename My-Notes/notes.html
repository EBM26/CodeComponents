Module # 1: Variables

  1. 'let' and 'cont' variables share in common the fact that they are blocked scoped, meaning they are only available inside a block of code defined by the curly brackets {}.
  2. The main difference between 'let' and 'const' variables is that 'let' variables can be updated and their values change where as 'const' variables are just as they sound, constant, they can NEVER be updated.
  3. RULE OF USE: This is a good practice to follow when writing varaiables in ES6:
  1. Never use 'var'.
  2. Use 'const' by default for all your variables.
  3. Use 'let' only if you need to change the value of your variable later on in your code.

  1. ES6 comes with 4 new string methods that allow you to inspect strings kind of like regex:

  2. startsWith( ) checks is a string starts with whatever characters you pass into the parentheses. It also takes a second argument that can start from a certain character number by passing in the numbers of characters to ignore. IS CASE SENSITIVE.
    3. const String1 = '20-AC2018-jz'
